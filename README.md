# bpo-bakaiti-hub

A full-stack, real-time BPO floor hub for agents. Features include:
- Real-time Gossip Tea Box (Anonymous confessions)
- Chai/Samosa Sponsor Spin-the-Wheel
- Shift & Lunch Polls
- Leaderboard & Awards (Wall of Fame & Shame)
- Voice Catchphrase Soundboard
- Escalation Quote Nominee Board

## How to run locally:
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open in browser: `http://localhost:3000`
